# peercli channel

The `peercli channel` command allows administrators to perform channel related
operations on a peer, such as joining a channel or listing the channels to which
a peer is joined.

## Syntax

The `peercli channel` command has the following subcommands:

  * fetch
  * getinfo
  * join
  * joinbysnapshot
  * joinbysnapshotstatus
  * list
  * signconfigtx
